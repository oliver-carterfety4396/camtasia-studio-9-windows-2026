## Usage

A common workflow is:

1. Launch the application.
2. Start a new recording or bring in existing footage.
3. Use the timeline to trim, reorder, and polish clips.
4. Add cursor highlighting or other visual refinements where appropriate.
5. Select export preferences, including resolution and language options.
6. Render the finished video and save it to your chosen location.

For capture sessions, it helps to run a short test recording first so you can verify microphone input, window focus, and display settings. For editing, place your media on the timeline early, then assemble the final sequence with cuts, annotations, and export settings as needed.
